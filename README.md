# @sasjs/electron

<!-- ALL-CONTRIBUTORS-BADGE:START - Do not remove or modify this section -->

[![All Contributors](https://img.shields.io/badge/all_contributors-1-orange.svg?style=flat-square)](#contributors-)

<!-- ALL-CONTRIBUTORS-BADGE:END -->

A desktop application that has imbedded `@sasjs\reat-seed-app` and `@sasjs\server` written in TypeScript.
`@sasjs\reat-seed-app` is running at `http://localhost:3000` by default (can be changed by editing `start:electron` script in `package.json`).
`@sasjs\server` is running at `http://localhost:5000`.
In order to start `@sasjs\server` `Node.js` has to be installed on targeted computer.
